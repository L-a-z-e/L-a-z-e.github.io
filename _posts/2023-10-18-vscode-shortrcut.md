# VS Code

- 이름 한번에 바꾸기 : shift + f6

- import를 알아서 가져오기: alt+shift+o

- 주석처리 : ctrl + /

- 영역을 주석처리 :ctrl + alt + /

- 터미널을 열기 : ctrl + shift + A

- 검색 : ctrl + f

- 윗줄 복사 붙여넣기 : Alt + Shift + 방향키 아래

- 매직키 : ctrl+space = 툴안에 있는 속성을 알려줌

- 줄바꿈 : ctrl + shift + 방향키 위,아래

- 줄지우기 : shift + del

- 매직키2 : alt + enter = 오류 뜰때 해결방법을 제시해줌
